JGammon - Fenster und Dialoge

Das Hauptfenster

Oben Menüleiste und Werkzeugleiste. Dann das eigentliche Spielfeld. Dort mit der Maus Steine bewegen. Vom Ausgangszacken (oder der Schranke) zum Zielzacken (oder dem Ausfeld). Wird ein Stein "in die Hand" genommen so werden die Möglichen Zielfelder in einer anderen Farbe und mit einem anderen Symbol gekennzeichnet. Die Werkzeugleiste ist frei beweglich und kann mit der Maus bewegt werden. Sie kann als eigenes Fenster angezeigt werden oder "angedockt" an das Hauptfenster, dann links, rechts, oben oder unten (empfohlen ist oben). Sie Aktionen, die mit einer Taste verknüpft ist, im Moment nicht verfügbar, so ist diese Taste deaktiviert.
